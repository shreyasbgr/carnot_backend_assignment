Carnot Backend Assignment Solution

Welcome to my solution for the Carnot Backend Assignment!

Code repository link: https://github.com/shreyasbgr/carnot_backend_assignment

Solution to all the sub-questions:

  1. URL for the Admin access to the database with the required data: https://carnot-backend-assignment.herokuapp.com/admin/ Username: admin Password: admin

  2. URL to get student detailed info is:
    https://carnot-backend-assignment.herokuapp.com/application/student/student_id
    where, student_id represents the id of the student.
    For example, the URL for student detailed info for student with student_id = 1 is,
    https://carnot-backend-assignment.herokuapp.com/application/student/1

  3. URL for the Student Search Form HTML page: https://carnot-backend-assignment.herokuapp.com/application/students/search

Additionally, to check the list of all students use the URL: https://carnot-backend-assignment.herokuapp.com/application/students/

The application uses the Python Django framework and is deployed on Heroku. The database used by the application is the free-tier Heroku Postgres database.
